Tabs - Direction

To align links, use the .pull-left or .pull-right utility classes. Both classes will add a CSS float in the specified direction.

  • Right
  • Right tab
  • left
  • left tab 
  <div id="direction" class="clearfix">

    <!-- Right Aligned -->
    <div id="right">
      <ul class="nav nav-tabs pull-right">
        <li>
          <a href="">Right</a>
        </li>
        <li class="active">
          <a href="">Right tab</a>
        </li>
      </ul>
    </div>
    <!-- /right-aligned -->

    <!-- Left Aligned -->
    <div id="left">
      <ul class="nav nav-tabs pull-left">
        <li class="active">
          <a href="">left</a>
        </li>
        <li>
          <a href="">left tab</a>
        </li>
      </ul>
    </div>
    <!-- /left-aligned -->

  </div>
