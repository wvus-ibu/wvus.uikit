Introduction

The World Vision UiKit is used for styling of elements and JavaScript widgets used across the World Vision sites.

File Structure

Within the UiKit, you can easily find the files to link in your site

Once downloaded, unzip the folder to see the structure of the kit. You will see this:

      wvus.uikit/
    ├── css/
    │   ├── wvus.uikit.all.css
    │   ├── wvus.uikit.all.min.css
    │   ├── wvus.uikit.core.css
    │   ├── wvus.uikit.core.min.css
    ├── js/
    │   ├──jquery.min.js
    │   ├── wvus.uikit.all.js
    │   ├── wvus.uikit.all.min.js
    │   ├── wvus.uikit.core.js
    │   ├── wvus.uikit.core.min.js
    ├── img/
    │   ├── {World Vision Images}
    │   └── 
    └── font/
        ├── FontAwesome.otf
        ├── fontawesome-webfont.eot
        ├── fontawesome-webfont.svg
        ├── fontawesome-webfont.ttf
        └── fontawesome-webfont.woff

This is the basic, out of the box form of the WVUS UiKit. We provide the compiled CSS and JS from Bootstrap and Font Awesome (core). Any other libraries that are included in the kit will be identified with extra in the file name. The all identifier includes core and extra in one CSS and JS file.

The img directory contains all images from Bootstrap, Font Awesome and World Vision. The font folder contains the default fonts included in Font Awesome

DO NOT modify the font folder or its contents. Doing so can cause Font Awesome to not work.

HTML Example

Including the World Vision UiKit is easy. Here's an example to prove it.

Here's a typical HTML File:

      <!DOCTYPE html>
      <html>
        <head>
          <title>WVUS UiKit Example</title>
          <meta name="viewort" content="width=device-width, initial-scale=1.0">
        </head>
        <body>
          <h1>Hello World!</h1>
          <script type="text/javascript" src="js/jquery.min.js"></script>
        </body>
      </html>

To make it UiKit compatible, just include the CSS and JS files:

    <!DOCTYPE html>
      <html>
        <head>
          <title>WVUS UiKit Example</title>
          <meta name="viewort" content="width=device-width, initial-scale=1.0">
          <!-- WVUS UiKit -->
          <link rel="stylesheet" type="text/css" href="css/wvus.uikit.all.min.css">
        </head>
        <body>
          <h1>Hello World!</h1>
          <script type="text/javascript" src="js/jquery.min.js"></script>
          <script type="text/javascript" src="js/wvus.uikit.all.min.js"></script>
        </body>
      </html>

And it's that easy! Just include the wvus.uikit.all.min.css and wvus.uikit.all.min.js in your HTML, and your on the way to developing a site with the World Vision brand.

The UiKit requires jQuery to be included before the wvus.uikit.all.min.js. We have included a version to use, js/jquery.min.js.
