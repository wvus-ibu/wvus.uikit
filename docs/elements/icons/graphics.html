SVG of sector icons. Easily sized and browser compatible. Icons are set to the background, so they will work with any element

The SVGs can be scaled to any size. Set the width and height in CSS to make the icons be as large or small as desired. There are also convenience sizes available.

Browser Compatibility: Modernizer.js, provided, will set a svg class on the html element when the broswer supports rendering of SVGs. It will set a nosvg class if the browser doesn't support rendering of SVGs. In that case, PNGs have been provided as fallbacks for the convenience sizes
