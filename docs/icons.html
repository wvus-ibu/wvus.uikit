How to use

All icons require an <i> tag with a unique class, prefixed with icon-. To use, place the following code just about anywhere:

          <i class="icon-search"></i>

Heads up! When using beside strings of text, as in buttons or nav links, be sure to leave a space after the <i> tag for proper spacing.

Larger Icons

To increase the size of icons relative to its container, use icon-large (33% incease), icon-2x, icon-3x, icon-4x classes.

Stacked Icons

Easily stack multiple icons on top of each other to create a singe custom icon. Use icon-stack class on the parent element and icon-stack-base for the bottom icon.

Note: The bottom icon must be the first icon in the HTML with the icon-stack-base class. 
          <span class="icon-stack icon-2x">
            <i class="icon-sign-blank icon-stack-base"></i>
            <i class="icon-signal icon-light"></i>
          </span>

Custom CSS

Font Awesome renders all icons as a font; therefore, any font styling can be applied to the icon.

          <i class="icon-bug icon-4x" style="color: green;"></i>
